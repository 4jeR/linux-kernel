Plan prezentacji

1. Szybkie wprowadzenie do jądra systemu (ok. 10-15 min) - Olek

a) co to jest jądro systemu?
b) funkcjonalność Jądra
c) rodzaje jądra

2. Zarządzanie pamięcią – (ok. 20min) - Bartek

a) model NUMA
b) pamięć fizyczna i logiczna
c) stronicowanie i tabele stronnicowe
e) slab layer, slab allocator

3. Procesy, przełączanie i zarządzanie procesami (ok. 30-40 min) - Olek

a) priorytety i przydział procesora
b) typy procesów (Kernel process/user process)
c) przekazywanie danych pomiędzy procesami
d) planer
e) startowanie, kopiowanie, usuwanie procesów

4. Wirtualny system plików (VFS) – (ok. 20-25min) - Bartek

a) typy systemów plików
b) powszechny model plików (Common File Model)
c) struktura VFS - obiekty: superblock object, inode object, file object oraz operacje na nich wykonywane
d) omówienie standardowych funkcji

5. Trochę o kompilacji jądra i modułach (ok. 10-15 min) - Olek

a) po co w ogóle kompilować samemu Kernel
b) trochę o modułach
c) proces kompilacji i ustawień configu

6. System calls – (ok. 20-25min) - Bartek

a) czym jest System Call?
b) komunikacja z jądrem systemu
c) typy System Calli - dostępne wywołania systemowe
